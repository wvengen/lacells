Use in µg Local-GSM-Backend

This data can be used for offline positioning on Android devices using Local-GSM-Backend and µg's UnifiedNLP.

Normally you would need to download the complete OpenCellID database, but this repository allows you to just pick the countries you're interested in. The helper script genlacells makes this easy. Download it, give it executable permissions (chmod +x genlacells) and run it with the desired MCC numbers as argument (separated by spaces). It will download the files from this websites, and generate a lacells.db file with up-to-date locations of cell towers for your country.

License

The data is aggregated from both Mozilla Location Service and OpenCellID. The license for these datasets are public domain and CC-BY-SA 3.0 respectively. By releasing this dataset under the terms of the CC-BY-SA 3.0 license, the terms of both source licenses are met.

In short: you have to mention OpenCellID when this dataset is used (but be nice and also mention Mozilla Location Service). And when you re-publish it, use CC-BY-SA 3.0.
